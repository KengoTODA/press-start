PRESS START!

Welcome to our site providing template for enchant.js! Our template helps you to create HTML5 game quickly.

1. choose plugins
2. download your Rakefile

download Rakefile

# please name it "Rakefile"

# copy below if you cannot download

3. start to code

execute rake command at the directory you saved Rakefile. All templates will be created. What you need is only editing game.js to create your game.

If you have r.js (JavaScript optimizer), you can execute rake optimize to optimize your codes. Optimized codes will be saved into optimize directory.

To create zip file to distribute, please execute rake package command. This zip file is 9leap ready. To package optimized game, please execute at optimize directory.

What is PRESS START?

PRESS START is a web site providing templates for enchant.js.

rake is required to use template. You can optimize your code if you have r.js.

What is rake?

rake is build tool. Please read Wikipedia.
